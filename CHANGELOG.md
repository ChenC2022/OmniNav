## [0.2.11] - 2026-03-12

### 新增

- **书签图标多路获取与自定义**：
  - **自动获取增强**：图标获取顺序为 Google S2 → 站点 `/favicon.ico` → 站点 `/apple-touch-icon.png` → 抓取页面 HTML 解析 `<link rel="icon">` → **主域名 fallback**（如 `dash.example.com` 失败时再试 `example.com`）。请求站点时使用浏览器 UA，减少被 Cloudflare 等返回挑战页导致拿不到图标的情况。
  - **自定义图标地址**：编辑/添加书签时可选填写「图标地址」（图片 URL），保存后该书签优先使用该地址显示图标，适用于无法自动拉取图标的站点（如受防护的子站）。列表与编辑弹窗内图标展示均支持自定义 URL，加载失败时自动回退到自动获取或首字母。
  - **编辑弹窗预览**：URL 下方图标预览在填写「图标地址」时优先显示自定义图标；支持「刷新图标」强制重新请求自动图标。

### 变更

- **BookmarkIcon**：展示顺序为自定义 URL（若已设置）→ 接口自动获取 → 站点 fallback → 首字母；书签或 `favicon` 变化时重置展示阶段。
- **BookmarkForm**：打开弹窗或修改「图标地址」时重置预览错误状态，避免沿用上一次失败结果。

## [0.2.10] - 2026-03-09

### 变更

- **天气显示增强**：天气组件展示「气象情况 + 温度」，温度显示 **1 位小数**。
- **天气代理与缓存**：`/api/weather` 通过 Open‑Meteo 获取数据，并使用 Cloudflare Cache API + KV 做 **15 分钟**缓存，减少上游请求与提升响应速度。
- **时间显示优化**：时间组件改为 `03月09日 星期一 15:21` 格式；页面加载/刷新时更新、之后按分钟更新，并在页面重新可见时立即同步。

## [0.2.8] - 2026-03-09

### 变更

- **搜索栏默认与交互**：
  - **默认书签搜索**：打开页面时搜索模式默认为书签搜索，可点击「网页/书签」或 Tab 切换。

## [0.2.7] - 2026-03-09

### 新增

- **跨分类拖放至分类名**：编辑布局下，可将书签拖到目标分类的**标题行**（分类名区域）松开，书签会移入该分类并插入到**第一位**；标题行悬停时有高亮反馈。

### 变更

- **0 级折叠高度**：折叠为「仅标题」时，内容区高度由 1rem 改为 0，与空分类视觉高度一致。

## [0.2.4] - 2026-03-07

### 变更

- **分类折叠「仅标题」**：内容区保留 1rem 高度（不再完全塌缩为 0），视觉更稳定。

### 修复

- **常用书签持久化**：修复「添加到常用」后刷新或重启页面常用书签消失的问题。
  - 原因：分类卡片内右键「添加到常用」时未注入 `savePinned`，仅更新了本地 store 未写入云端。
  - 修改：在 `CategoryCard.vue` 中注入 `savePinned`，确保常用列表变更后同步至 KV。
  - 同时：仅在书签接口加载成功时执行「清理常用中失效 ID」，避免书签请求失败时误将常用清空并写回；登录/设密成功后调用 `loadData` 再跳转，保证进入首页时书签与常用数据正确加载。

## [0.2.3] - 2026-03-06

### 新增

- **分类折叠功能**：支持 4 级展开深度调整（最小形态/较小/较大/全部展开），状态自动同步至云端。针对「未分类」区域优化了折叠逻辑，跳过「仅标题」态以防止内容完全不可见。
- **AI 快速添加**：常用区与分类区之间新增快速粘贴栏，粘贴 URL 后由 AI 自动生成标题、描述并归入分类，确认后一键入库；支持多 URL 批量添加与深度网页分析。
- **书签编辑增强**：书签表单支持手动选择所属分类；AI 生成信息时会自动建议最合适的分类并自动选中。
- **UI 细节优化**：
  - **回到顶部**：新增透明磨砂质感的悬浮按钮，在滑过「常用区域」后按需出现，配色与顶部栏按钮对齐。
  - **视觉统一**：统一了书签编辑/添加表单的按键配色（使用主题靛蓝色）与加载状态。
  - **分类统计**：分类卡片右上角新增书签数量统计（非悬停时显示）。
  - **AI 对话**：侧边栏新增「刷新」按键，支持一键清空上下文并开启新会话。

### 变更

- **UI 布局重构**：
  - 「拖动书签」全局开关移至顶部导航栏，带垂直分隔线设计。
  - 分类设置和未分类操作按键统一左移至标题旁，页面重心更稳。
  - 常用区网格自适应增强，移动端窄屏支持一行 3 个书签布局。
- **数据结构扩展**：`Category` 类型新增 `viewLevel` 字段，用于持久化 UI 展开状态。


### 变更

- **界面布局调整**：搜索栏移动至顶部导航栏（桌面端）或页面最上方（移动端），操作更直观。
