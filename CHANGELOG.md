# 更新日志

## [0.2.4] - 2026-03-07

### 变更

- **分类折叠「仅标题」**：内容区保留 1rem 高度（不再完全塌缩为 0），视觉更稳定。

### 修复

- **常用书签持久化**：修复「添加到常用」后刷新或重启页面常用书签消失的问题。
  - 原因：分类卡片内右键「添加到常用」时未注入 `savePinned`，仅更新了本地 store 未写入云端。
  - 修改：在 `CategoryCard.vue` 中注入 `savePinned`，确保常用列表变更后同步至 KV。
  - 同时：仅在书签接口加载成功时执行「清理常用中失效 ID」，避免书签请求失败时误将常用清空并写回；登录/设密成功后调用 `loadData` 再跳转，保证进入首页时书签与常用数据正确加载。

## [0.2.3] - 2026-03-06

### 新增

- **分类折叠功能**：支持 4 级展开深度调整（最小形态/较小/较大/全部展开），状态自动同步至云端。针对「未分类」区域优化了折叠逻辑，跳过「仅标题」态以防止内容完全不可见。
- **AI 快速添加**：常用区与分类区之间新增快速粘贴栏，粘贴 URL 后由 AI 自动生成标题、描述并归入分类，确认后一键入库；支持多 URL 批量添加与深度网页分析。
- **书签编辑增强**：书签表单支持手动选择所属分类；AI 生成信息时会自动建议最合适的分类并自动选中。
- **UI 细节优化**：
  - **回到顶部**：新增透明磨砂质感的悬浮按钮，在滑过「常用区域」后按需出现，配色与顶部栏按钮对齐。
  - **视觉统一**：统一了书签编辑/添加表单的按键配色（使用主题靛蓝色）与加载状态。
  - **分类统计**：分类卡片右上角新增书签数量统计（非悬停时显示）。
  - **AI 对话**：侧边栏新增「刷新」按键，支持一键清空上下文并开启新会话。

### 变更

- **UI 布局重构**：
  - 「拖动书签」全局开关移至顶部导航栏，带垂直分隔线设计。
  - 分类设置和未分类操作按键统一左移至标题旁，页面重心更稳。
  - 常用区网格自适应增强，移动端窄屏支持一行 3 个书签布局。
- **数据结构扩展**：`Category` 类型新增 `viewLevel` 字段，用于持久化 UI 展开状态。


### 变更

- **界面布局调整**：搜索栏移动至顶部导航栏（桌面端）或页面最上方（移动端），操作更直观。
- **励志语位置更新**：励志语从顶部栏移至「常用」区标题旁，增加 ✨ 图标并居中展示，设计感更强。
- **品牌形象升级**：顶部栏和登录页引入全新 `logo.svg` 定制图标，提升项目辨识度。
- **代码重构**：将励志语逻辑提取为 `useQuotes` composable，实现兄弟组件间的状态共享，代码更易维护。

## [0.2.1] - 2026-03-05

### 修复

- **AI 助手 500 错误**：补充缺失的 `getJson` 辅助函数，修复从 KV 读取 settings 时的 ReferenceError
- **favicon 502**：获取失败时返回透明占位图而非 502，减少控制台报错

### 变更

- 移除 `wrangler.toml`，所有配置（含 KV 绑定）在 Cloudflare Dashboard 中完成，便于新用户使用 Bindings 的 Add 按钮

- 本地开发默认与生产一致，采用「首次使用，请设置密码」；新增 `.dev.vars.example`
- `.cursor` 目录加入 `.gitignore`，不再同步到 GitHub
